# 散散步

六月的某天傍晚，散步后回实验室写下：

这是一个“心随境转”和“境随心转”的问题。

当身处困难时期的时候，心里总是想着等这个困难解决，有了自己满意的结果，自己就可以轻松下来了。
但是呢，随着年岁增长，这些悬而未决的难题，是越来越多的，它们可能中间存在空隙，让人得到些许闲适与轻松，但是这些空隙其实是很难得的，对目前的我来说。

像昨天结束的高考，对于他们，可能度过高考，就迎来一段空隙，但是我已经明白，这种空隙在以后的日子里，会越来越难得，以后的日子往往伴随着更多的难题，它们会一件接一件来，甚至好几件一起来，可能短暂停留，也可能长期停留。

所以，究竟是要“心随境转”，困难存在，我就严阵以待，心里期盼着解决的那一天，就可以轻松了，就可以不焦虑不苦闷了；还是“境随心转”，无论生活中是否有难题，都保持一种长期温和的心态，不因为境遇而改变自己的心态。

前者，是人之常情吧，至少我是如此。后者呢，是我希望自己可以达到的境界，只是知易行难，始终做不到。

从去年十二月份开始，那时候小论文几次被拒之后第一次送审，到今年四月份真正接收，三月份开始找暑期实习，到五月份从一家公司干半月离职，到六月份也没有新的机会，以为有的退路和保底今年也受 AI 冲击了无音讯。困难时期一直没有结束，小论文接收后因为拖了这么多个月，也没有明显的喜悦之情，依旧沉溺在其他的苦闷中，并且这种苦闷会一直持续到秋招，持续到我真的拿到一个自己满意的 offer 吧，那得是什么时候了，十一月？十二月？

我一直在期盼许多个自己满意的结局，期盼着有足够大的空隙让我透透气，可是事情总是不尽如人愿，旧的苦难结束，新的苦难回来，中间没有空隙，甚至会叠加在一起。

在研究生开始的时候，我写下“一场新的修行”，我现在理解了，这场修行，就是“心随境转”到“境随心转”的修行。经历这么多悬而未决的事情，经历过许多情绪的阴晴变化，其实都是这个课题。

不过我觉得吧，虽然说苦难一直没有结束，一件接一件，但是我现在的心态已经平静多了，没有那么焦虑和苦闷，大多数时候还是可以认真去做事的，也可以进入心流状态，只是闲下来的时候，会有一些空虚趁虚而入，我去散散步，自己和自己说说话，算是一种排解。

我现在觉得，这场修行，没有答案，没有边界（最近学 Agent，很重要的就是边界，比可以做什么更重要的是不可以做什么），没有评估标准（你无法知道自己做到了吗，人类心里的东西大多数都没有办法量化，没有办法评测吧，不像一个 Agent 系统那样）。

关于这一点，自己做的好不好，心里会有中隐隐的感觉，但我说不清楚。可是我觉得经历的苦难，如果能在其中寻求到些许存在的意义，就算有进步吧。

# 聊聊天

与 Claude 聊聊之后，有种豁然开朗的感觉。

我似乎把「境随心转」本身也当成了一个待解决的难题——「等我修行到那个境界，就不焦虑了」。并且「境随心转」大概不是抵达之后就常驻的状态，心的修行，不是一个有完成态的任务。

没有评估标准也不尽然，空虚来临时出门散步排解、自己和自己说说话、沉下心来进入心流状态，这些其实都算空隙，是我自己凿出来的。困难仍然在，境没变，但是自己凿出了空隙，所以并不像我想的那样，真正解决之后空隙才会出现。心里的事情无法量化，但这些小空隙确实存在，只是无法汇成一个分数而已。

面对困难有苦闷情绪是对现实的正常反应，修行修的不是「不苦」，而是苦的时候，那些自己凿出的空隙的大小。

并且，苦难解决之后，也不是想象中「真实的更大的空隙」。譬如四月小论文接收后，喜悦被漫长周期稀释，新的苦闷已经叠加上来。还有一层：大空隙未必更好过，前面提到的高考结束，虽然轻松确实存在，但茫然也确实存在，我是用现在的视角去看过去，本就不是一个平等的视角，毕竟人无法同时拥有青春和对青春的感受。

问题是在我心中，我把自己凿出的小空隙标注为“这不算数，真正的休息在后面”，而往后呢，问题会叠着来，如果一直抱着这样的心态，我几乎就没有合乎心意的休憩了。

并且，小空隙才是真实存在、真实发生的，大空隙是很难得的，我渴望后者，贬值前者，有种莫名的偏执。

所以，这场修行，不是「心随境转」到「境随心转」（太空泛且达不到），而是在空隙来临时，淡然愉悦地度过，在哪个空隙里，就完整地待在哪个空隙里——这可能就是平衡本身。

最后小克和我讲：**你现在能说得这么透，不全靠想通了，也因为此刻刚好身处一个空隙里。等哪天苦闷重新漫上来、这些话一个字都想不起来的时候——那不是退步，那才是这场修行真正开始的地方。眼下这份清明，也只是又一次预习。**
